# TCC-ELETRO
An intelligent industrial monitoring system that uses machine learning to identify individual equipment consumption in real time via Arduino sensors. It detects motors, compressors and pumps through non-intrusive load monitoring, providing energy analysis, anomaly detection and optimization recommendations through an intuitive interface.

-Português

Este Software foi desenvolvido para ser complementado a seu dispositivo de medição físico. Desenvolvido majoritariamente em linguagem Python e com assistência de IA, o mesmo busca a comunicação entre medidor e usuário de forma inteligente, eficiente e barata. Medidores analógicos e/ou digitais presentes no mercado atual custam caro, com valores podendo chegar a 10.000 reais até 100.000 reais, além do preço ser demasiadamente alto, estes, não são instalados de forma fixa, são utilizados periodicamente e exigem a interpretação de um engenheiro ou técnico. Este projeto surge com o intuíto de viabilizar este medidor de forma fixa na indústria, para que o cliente tenha acesso 24/7 a condição de sua rede elétrica. Com um software submetido a bibliotecas de equipamentos elétricos, como a NILM, foi treinado para identificar falhas e distúrbios na rede elétrica se baseando em medições de potência, corrente, temperatura, tensão, fator de potência, reativos e outros. Vale ressaltar que não fora criado uma Inteligência Artificial do zero, utilizamos a API do Google Gemini e implementamos um algoritmo linkado à um arquivo .JSON que possui em sua memória dados de todos os equipamentos já submetidos ao sensor.

-English

This software was developed to complement your physical measurement device. Developed primarily in Python with AI assistance, it seeks to establish communication between the meter and the user in an intelligent, efficient and low-cost way. Analog and/or digital meters currently available on the market are expensive, with prices ranging from 10,000 reais to 100,000 reais. Besides the excessively high cost, these devices are not permanently installed, are used periodically and require interpretation by an engineer or technician. This project arises with the purpose of making this meter feasible for permanent installation in industry, allowing customers to have 24/7 access to their electrical grid condition. With software submitted to electrical equipment libraries such as NILM, it was trained to identify faults and disturbances in the electrical grid based on measurements of power, current, temperature, voltage, power factor, reactive power and others. It is worth noting that an Artificial Intelligence was not created from scratch; we used the Google Gemini API and implemented an algorithm linked to a .JSON file that stores in its memory data from all equipment already submitted to the sensor.
